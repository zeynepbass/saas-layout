
- **Page / Layout (UI):** Handles user interface and rendering. Components, pages, and layouts live here.  
- **Hook (Client logic):** Contains custom hooks and client-side logic for state management or interactions.  
- **Service (Business logic):** Handles business rules, data processing, and communication with lower-level services.  
- **Lib (Low-level fetch):** Contains reusable functions for API calls or utilities.  
- **External API:** Third-party APIs or backend services that provide data.
